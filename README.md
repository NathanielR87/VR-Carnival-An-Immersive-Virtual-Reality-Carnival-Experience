## Overview

VR Carnival is a fully interactive virtual reality carnival game developed as an individual project to explore immersive application development using Unity.

The project challenged me to learn an entirely new development environment while designing intuitive VR interactions, implementing real-time physics, and creating an engaging user experience.

Although originally developed for Google Cardboard, the project was designed with the potential to expand to additional VR platforms.

---

## Motivation

I have always been fascinated by Virtual Reality and believe immersive technologies will play an important role in the future of education, entertainment, training, and human-computer interaction.

This project was my opportunity to teach myself Unity and begin exploring VR software development from the ground up.

---

## Features

- Immersive VR gameplay
- Physics-based ball throwing
- Interactive carnival environment
- Bottle knockdown mechanics
- Score tracking
- VR camera controls
- Collision detection
- Real-time interaction

---

## Technologies

- Unity

---

## Architecture

The project follows a modular object-oriented architecture where gameplay systems are separated into independent components responsible for player interaction, object physics, game logic, scoring, and camera control.

This organization made it easier to debug individual systems while keeping the project maintainable as additional features were added.

---

## Engineering Challenges

The most difficult aspect of this project was creating intuitive VR interactions.

Challenges included:

- Camera tracking
- Player orientation
- Object interaction
- Motion responsiveness
- Physics tuning

Designing gameplay that felt natural inside VR required significantly different thinking than traditional desktop applications.
